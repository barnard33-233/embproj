# embproj

## 简介

这是一个 stm32f4xx 的嵌入式工程项目。使用的软件为 Keil 5。是嵌入式的结业实验。

功能是放歌。有下面的按键功能:

- A 键暂停，B 键播放
- 可变速，\# 键开始输入速度，然后按数字键写入速度，再次按下 \# 提交速度
